# AI-Powered CSV Dashboard

Upload your CSV, get smart KPIs, visualizations, and AI-generated summaries instantly!

## Tech Stack
- React + Tailwind CSS
- Node.js + Express
- MongoDB
- OpenAI API
- Netlify + Render

## Features
- 🔐 Login/Signup Auth
- 📁 CSV Upload
- 📊 Dynamic KPI + Chart Gallery
- 🧠 AI Summary with GPT
- 📤 Export as PDF/Image

## Setup
1. Clone the repo
2. Add `.env` 
3. Run frontend and backend separately
